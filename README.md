# Text Analyzer (Python)

## Description
This is a simple Python mini project that analyzes a given text
and provides:
- total number of words
- number of unique words
- most frequent word

## Example Input
Data science is fun and data science is powerful

## Example Output
Total words: 9  
Unique words: 6  
Most frequent word: data  

## Concepts Used
- Strings
- Lists
- Dictionaries
- Functions
- Basic class structure

## Purpose
This project is part of my Python learning journey to build
strong fundamentals before moving to Data Science and AI.

Version 1 – Basic text analyzer
